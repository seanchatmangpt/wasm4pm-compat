```gitignore
# Residents: gap-ledger.yaml, *.md (indices, reports), *.yaml (feature maps, ledgers),
# *.rs (generated stubs), *.wit (component interface), *.sh (audit scripts)

ggen/emitted/*.md

ggen/emitted/*.wit

```

**Rationale:** These files are **regenerated** by orchestration; they are not handwritten source and do not carry project intent. Committing them creates noise in diffs and violates the SoT (source of truth) principle: the true source is the input intel, rules, templates, and projections.

## Artifact Lifecycle

### Generation Trigger

The ggen orchestrator runs on:
1. Changes to `ggen/intel/` (new intelligence discovered)
2. Changes to `ggen/rules/` (law amendments)
3. Changes to `ggen/templates/` (generation logic)
4. Changes to `ggen/projections/` (new projection targets)
5. Manual invocation: `ggen generate` (or equivalent)

### Generation Flow

```
ggen/intel/
  ├── *.md (research, intelligence)
  └── *.yaml (ledgers, capability maps)
ggen/rules/
  └── *.yaml (law declarations)
ggen/templates/
  └── *.tera (code generation)
ggen/projections/
  └── *.yaml (projection config)
              ↓
      [ggen orchestrator]
              ↓
  ggen/emitted/
  ├── gap-ledger.yaml
  ├── GAP_LEDGER_REPORT.md
  ├── *.rs (generated stubs — future)
  ├── *.wit (component interface — future)
  └── *.sh (audit scripts — future)
```

### Review Workflow

1. **Orchestrator runs** → emitted files regenerated
2. **Human reviews** emitted artifacts (YAML + markdown)
3. **If closure conditions met** → merge or approve
4. **Emitted files discarded** before commit (never committed to main)

## Closure Conditions & Audit Gates

Each gap has a **closure condition** (what must be true) and an **audit gate** (how to verify):

| Gap | Closure Condition | Audit Gate |
|-----|-------------------|-----------|
| **GAP_001** | compat → wasm4pm type bridge; witnesses preserved; refusal laws aligned | Integration tests: OCEL → discovery → PetriNet → conformance → receipt |
| **GAP_COMPONENT** | Component Model WIT interfaces generate for all feature gates; witness encoding valid | WIT syntax valid; wit-bindgen generates trait Guest; conformance pass |
| **GAP_LOSS** | Loss policies auto-detected; LossReport emitted on all lossy projections | All format conversions carry named policies; loss accounting trace complete |
| **GAP_PROCESS_TREE** | Process tree arity, POWL soundness, projection legality enforced at compile-time | Compile-fail fixtures prove wrong arity, invalid projections are rejected |
| **GAP_TS** | TypeScript .d.ts surfaces generated from Rust types via specta; zero-cost phantom encoding | audit-no-dto-flattening.sh passes; specta codegen produces valid .d.ts |
| **GAP_WASM** | WASM boundary memory-safe and type-law-respecting; wasm-bindgen bindings generated | wasm-pack build succeeds; WASM boundary validates against prohibited list |

All gaps currently marked **MANUFACTURED** as of 2026-06-01.
